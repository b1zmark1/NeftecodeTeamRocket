# Attach Package For Organizers

Этот пакет подготовлен под формулировку организаторов:
- данные уже приведены к train-like raw-формату;
- остаётся только преобразование названий компонентов к типу компонента и преобразование массовых долей.

Файлы:
- `daimler_mixtures_train_patent_attach.csv`
- `daimler_component_properties_patent_attach.csv`

Что включено:
- только 2 сценария из `US20250136889A1` (`Oil W`, `Comparative Oil X`), потому что в них результат окисления дан в `A/cm`, что соответствует целевому столбцу хакатона;
- свойства антиоксидантов и базовые свойства некоторых патентных компонентов, которые можно согласованно представить в формате component properties.

Что не включено в attach-train:
- `US11142719B2` как train-scenarios не включён, потому что там окисление дано как `PAI`, а не `A/cm`, то есть целевая метрика не совпадает с хакатонной.

Ключевые допущения:
- `Дозировка катализатора, категория` выставлена в `1` для обоих патентных сценариев.
Причина: в патенте дан абсолютный уровень катализатора `100 ppm Fe`, тогда как в хакатоне используется только обезличенная категория `1/2`.
В исходном train категория `1` доминирует, в частности и для сценариев с биотопливом `5%`.
- Не полностью раскрытый остаток рецептуры из `Table 10` агрегирован в отдельный компонент `Прочее_patent_common_additives_1`.
Это честно сохраняет сумму массовых долей `100%`, не притворяясь известным конкретным компонентом.
